# 🌿 PlantCare AI

AI-powered plant disease detection system using **MobileNetV2 and Deep Learning**.

This project detects plant diseases from leaf images using a trained convolutional neural network and provides instant predictions through a web interface.

---

## 🚀 Features

- Upload plant leaf images
- AI-based disease detection
- Supports **38 plant disease classes**
- Built using **Transfer Learning with MobileNetV2**
- Web interface built using **Flask**
- Simple and user-friendly interface

---

## 🧠 AI Model

The model uses **MobileNetV2**, a lightweight deep learning architecture designed for efficient image classification.

### Model Details
- Input size: **224 × 224**
- Dataset: **87,000+ images**
- Classes: **38 plant diseases**
- Framework: **TensorFlow / Keras**
- Validation Accuracy: **~96%**
